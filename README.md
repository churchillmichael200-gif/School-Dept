# Computer Science Department Website

A modern, responsive website for the Computer Science Department featuring interactive design, smooth animations, and mobile-first approach.

## Features

### 🎨 Modern Design
- Clean, professional layout with modern typography
- Gradient backgrounds and subtle animations
- Card-based design for better content organization
- Responsive grid layouts

### 📱 Responsive Design
- Mobile-first approach
- Adaptive navigation menu
- Flexible grid systems
- Optimized for all screen sizes

### ⚡ Interactive Elements
- Smooth scrolling navigation
- Mobile hamburger menu
- Hover animations and transitions
- Active section highlighting
- Scroll progress indicator

### 🚀 Performance Optimized
- CSS custom properties for consistent theming
- Efficient animations with CSS transforms
- Lazy loading capabilities
- Throttled scroll events

## Technologies Used

- **HTML5**: Semantic markup with proper accessibility
- **CSS3**: Modern styling with Flexbox, Grid, and CSS Variables
- **JavaScript (ES6+)**: Interactive functionality and DOM manipulation
- **Font Awesome**: Icon library for visual elements
- **Google Fonts**: Inter font family for modern typography

## Getting Started

1. Open `Dept.html` in your web browser
2. The website is fully self-contained with no external dependencies required
3. All assets are loaded from CDNs for optimal performance

## Customization

### Colors
Modify CSS custom properties in `:root` to change the color scheme:

```css
:root {
    --primary-color: #your-color;
    --accent-color: #your-color;
    /* ... */
}
```

### Content
Update the HTML content in `Dept.html` to modify:
- Department information
- Course details
- Lecturer profiles
- Contact information

### Functionality
Extend `Dept.js` to add:
- Form validation
- API integrations
- Additional animations
- User authentication
